# Data Folder

This folder contains the datasets used for fraud detection model training and testing.

## Files

- fraudTrain.csv → Training dataset
- fraudTest.csv → Testing dataset

## Source

Dataset downloaded from Kaggle fraud detection dataset.

## Usage

The training script uses fraudTrain.csv to train the machine learning model.
The test dataset fraudTest.csv can be used for evaluation.

Place both files in this folder before running the training script.
